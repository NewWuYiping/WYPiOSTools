## Installation

WYPiOSTools is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'WYPiOSTools'

fastlane project:'WYPiOSTools' version:'0.2.4'
```
